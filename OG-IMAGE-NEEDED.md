# OG Image Required

For optimal SEO, you need to create an Open Graph image:

## Requirements:
- **File name**: `og-image.jpg`
- **Dimensions**: 1200px x 630px
- **Format**: JPG or PNG
- **Max size**: 8MB (recommended < 300KB)
- **Location**: `/public/og-image.jpg`

## Content Suggestions:
- SmartFlow Africa logo
- Tagline: "AI Automation for Nigerian Businesses"
- Visual: Modern tech graphics, Nigerian colors
- Include: WhatsApp icon, automation symbols

## Tools to Create:
- Canva (recommended): https://www.canva.com/create/facebook-posts/
- Figma
- Photoshop

## How to Add:
Once you create the image, save it as `og-image.jpg` in the `public` folder. The metadata is already configured to use it.
